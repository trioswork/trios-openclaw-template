# MEMORY.md, Memory Index

Use this as the index for private memory files created after installation.

Suggested structure:

```text
memory/
├── context/
│   ├── decisions.md
│   ├── lessons.md
│   ├── people.md
│   └── business-context.md
├── projects/
├── sessions/
├── integrations/
├── feedback/
└── pending.md
```

Do not put secrets in memory files.
